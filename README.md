# Fraud-Detection-Random-Forest
Random Forest Fraud Detection Model  | Kaggle Credit Card Dataset
# Credit Card Fraud Detection (Random Forest)

**Model Performance**  
- **Matthews Correlation Coefficient**: 0.8573  
- **Precision (Fraud)**: 97.37%  
- **Recall (Fraud)**: 75.51%  
- **F1-Score**: ~85.1%  
- **Accuracy**: 99.96% (imbalanced dataset)

**Dataset**: Kaggle Credit Card Fraud Detection (~57k normal + 98 fraud transactions)

**Key Highlights**:
- Extremely low false positives (only 2)
- Built with scikit-learn RandomForestClassifier
- Handled severe class imbalance

Tech Stack: Python, Pandas, Scikit-learn, Matplotlib, Seaborn
